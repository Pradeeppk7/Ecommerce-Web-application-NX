## Development server

Run `nx serve mykart` for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.

## Understand this workspace

Run `nx graph` to see a diagram of the dependencies of the projects.

## Creating an app

nx generate @nrwl/angular:app admin


## Creating components
 nx g component homepage --project=mykart --dry-run

## Remote caching

Run `npx nx connect-to-nx-cloud` to enable [remote caching](https://nx.app) and make CI faster.
